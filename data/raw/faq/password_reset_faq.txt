Password Reset FAQ

Purpose
This document provides basic information about password reset procedures at Nova Tech College.

When a Reset May Be Needed
A password reset may be needed if:
- the password is forgotten
- the account is locked
- login attempts have failed repeatedly
- the password has expired
- there is a suspected security issue

General Procedure
Students should use the official password reset process provided by the school. This may involve:
- identity verification
- confirmation through a recovery method
- creation of a new password

Security Advice
Students should:
- choose a strong password
- avoid reusing passwords from other platforms
- never share login credentials
- change the password immediately if misuse is suspected

Help and Support
If the reset process does not work, the student should contact the ICT helpdesk and provide student identification details where required.

Contact
Questions about password reset should be directed to the ICT helpdesk.
